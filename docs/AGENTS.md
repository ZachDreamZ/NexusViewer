# docs

## Purpose
Long-form documentation, screenshots, and reference images for the NexusViewer app.

## Ownership
Agent

## Local Contracts
- Keep total folder size modest (a few MB). Screenshots are PNG, optimized but not aggressively compressed.
- The README `## Screenshots` section is the single source of truth for what gets rendered publicly — anything in `screenshots/` should be referenced from there.

## File-specific contracts

### screenshots/
- `welcome.png` — first-run / no-project state
- `editor.png` — editor with sample markdown loaded
- `about.png` — About modal open
- All three are 1280×820 PNGs generated by `scripts/capture-screenshots.cjs`. Re-run that script (after `npm run build`) to refresh them.

## Verification
- `README.md` "Screenshots" section links resolve to files that exist on disk.
